🧱 Camadas da Aplicação

Pontos de Entrada

🚀
server.ts

API HTTP — porta 3003

Ponto de entrada da API. Inicializa o Fastify, registra plugins (CORS, JWT, Swagger), define hooks de autenticação e registra todos os grupos de rotas com prefixo /api.

@fastify/cors @fastify/jwt @fastify/swagger onRequest hook
⚙️
worker.ts

Processo separado — pm2

Processo independente que inicia os workers BullMQ e o RecurringJobScheduler. Consome jobs das filas Redis, executa webhooks HTTP e grava resultados no PostgreSQL.

BullMQ Worker processJob() Scheduler 30s Graceful shutdown

Camada de Libs / Infra

🔐 auth.ts

Valida JWT via JWKS, verifica permissões (10 escopos), sincroniza usuário/org com auth service, grava audit log.

⚡ bullmq.ts

getQueue(orgId, queueName) — factory que retorna a instância BullMQ para a fila correta, prefixada por organização.

🚨 errors.ts

AppError, NotFoundError, ValidationError, ForbiddenError. Mapeados para status HTTP no handler global.

🛡️ ip-allowlist.ts

Verifica se o IP do request está na lista de CIDRs/IPs permitidos configurada na API key. Bloqueia com 403 se falhar.

📝 logger.ts

Exporta instância pino configurada. Nível via LOG_LEVEL env. Pretty em dev, JSON em produção.

🗄️ prisma.ts

Singleton do PrismaClient. Pool configurado via DATABASE_URL. Reutilizado em todos os services.

🪣 rate-limiter.ts

Token bucket por API key implementado com script Lua atômico no Redis. Retorna 429 ao esgotar quota.

🔴 redis.ts

Exporta dois clientes ioredis: redis (geral, cache, locks) e bullRedis (conexão dedicada ao BullMQ).

🕐 scheduler.ts

RecurringJobScheduler — tick a cada 30s, adquire lock Redis de 60s, despacha jobs com nextRunAt <= now.

🔭 sentry.ts

Inicializa Sentry com DSN via env. Captura exceções não tratadas no server e no worker. tracesSampleRate configurável.

💡 Exemplos de Chamada de API

Criar um job

// POST /api/jobs // Header: Authorization: Bearer <token> const res = await fetch('https://job-assync-api.ddw1sl.easypanel.host/api/jobs', { method: 'POST', headers: { 'Content-Type': 'application/json', 'Authorization': `Bearer ${token}`, }, body: JSON.stringify({ queue: 'email-notifications', payload: { to: 'user@example.com', subject: 'Hello' }, callbackUrl: 'https://myapp.com/webhooks/jobs', maxAttempts: 3, scheduledAt: null, // null = imediato }), }); const job = await res.json(); // { id: 'clxyz...', status: 'PENDING', createdAt: '...' }

Criar jobs em lote (bulk)

// POST /api/jobs/bulk — até 1000 jobs const res = await fetch('.../api/jobs/bulk', { method: 'POST', headers: { /* ...auth headers */ }, body: JSON.stringify({ jobs: [ { queue: 'resize', payload: { file: 'img1.png' } }, { queue: 'resize', payload: { file: 'img2.png' } }, // ... até 1000 ], }), }); const { created, failed } = await res.json(); // { created: 2, failed: 0, ids: ['...', '...'] }

Criar job recorrente

// POST /api/recurring-jobs const res = await fetch('.../api/recurring-jobs', { method: 'POST', headers: { /* ...auth */ }, body: JSON.stringify({ name: 'Daily Report', queue: 'reports', cronExpression: '0 8 * * *', // todo dia às 8h payload: { type: 'daily' }, timezone: 'America/Sao_Paulo', maxRuns: 365, // opcional }), });

Consultar estatísticas

// GET /api/stats const stats = await fetch('.../api/stats', { headers: { 'Authorization': `Bearer ${token}` }, }).then(r => r.json()); // { total: 1024, pending: 12, running: 3, // completed: 998, failed: 11, queues: 4 } // GET /api/stats/timeline?hours=24 const timeline = await fetch('.../api/stats/timeline?hours=24', { headers: { 'Authorization': `Bearer ${token}` }, }).then(r => r.json()); // [{ hour: '2026-03-20T08:00Z', completed: 42, failed: 1 }, ...]
