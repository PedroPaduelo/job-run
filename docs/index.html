JobSync — Sistema de Fila Assíncrono

Plataforma de processamento assíncrono de jobs com multi-tenancy, filas, reprocessamento automático e webhooks. Backend Fastify + Prisma + BullMQ, Frontend React + Vite + Tailwind.

Node.js Fastify PostgreSQL Redis BullMQ React TypeScript

🏛️ Componentes do Sistema

🔧

Backend API

API REST Fastify, autenticação JWT + API Keys, workers BullMQ, scheduler de jobs recorrentes.

Porta: 3003
URL: https://job-assync-api.ddw1sl.easypanel.host
Docs: /docs (Swagger)
💻

Frontend

React + Vite, painel de controle, gerenciamento de filas, monitoramento de jobs em tempo real.

Porta: 5173
Build: estático (dist/)
Theme: Dark mode default
🗄️

PostgreSQL

Banco relacional multi-tenant com Prisma. Dados de usuários, organizações, filas, jobs e logs.

Host: 217.216.81.188
Porta: 54321
Credenciais no .env
⚡

Redis Broker

Broker para BullMQ, cache de sessão, locks distribuídos usados pelo scheduler.

Cache de JWKS (5min)
Filas BullMQ por organização
Lock de scheduler (60s)
🔐

Auth Service

Serviço de autenticação externo. Fornece JWT, perfis de usuário, organizações e membros.

URL configurável via AUTH_SERVER_URL
Sincroniza usuários e organizações
Ponto único de falha
⚙️

Workers

Processos BullMQ que consomem jobs das filas e executam webhooks com retry automático.

Concorrência configurável por fila
Timeouts e backoff exponencial
Callback opcional pós-execussão
