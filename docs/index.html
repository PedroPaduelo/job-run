Sobre este Projeto

O JobSync é uma plataforma centralizada para processamento assíncrono de jobs via webhooks. Construída com Fastify, React, BullMQ e PostgreSQL.

Esta documentação cobre de ponta a ponta: arquitetura, fluxos de dados, integrações, frontend e operações.
