Backup & Restore Procedures

pg_dump, Redis RDB/AOF, testes de restore

🗄️ O que Backupar

  • PostgreSQL: todos schemas
  • Redis: dump.rdb ou AOF
  • .env: secrets (manualmente, encriptado)

📦 PostgreSQL Backup

pg_dump -h $DB_HOST -p $DB_PORT -U $DB_USER $DB_NAME > backup-$(date +%F).sql
pg_dump ... | gzip > backup-$(date +%F).sql.gz
zcat backup.sql.gz | psql -h ... -U ... $DB_NAME

📦 Redis Backup

redis-cli BGSAVE
# Restore: copy dump.rdb + restart Redis

✅ Recomendações

  1. Automated daily backups — cron + S3
  2. Retention 30 dias
  3. Testar restore mensalmente
  4. Encriptar backups

🔍 Checklist

  • Backup diário configurado? ❌
  • Retention policy? ❌
  • Teste de restore regular? ❌
  • Backups encriptados? ❌
  • Off-site copy? ❌
