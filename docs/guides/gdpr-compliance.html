GDPR Compliance

Direito ao esquecimento, exportação, consentimento

🇪🇺 Requisitos GDPR

  • Direito ao esquecimento — deletar dados pessoais
  • Direito à portabilidade — exportar dados em JSON/CSV
  • Consentimento — registrar opt-in
  • Minimização — coletar apenas necessário

🔍 Aplicação ao JobSync

JobSync processa jobs payloads que podem conter PII (emails, CPFs, etc).

✅ Recomendações

  1. Data Deletion API — endpoint para deletar todos jobs de um usuário (por userId): 
    DELETE /api/users/:userId/jobs
  2. Data Export API — endpoint para exportar todos jobs de um usuário: 
    GET /api/users/:userId/export?format=json
  3. PII detection — scanner automático em payload/metadata para identificar PII
  4. Retention limits — auto-delete após X anos
  5. Consent log — auditoria de quando usuário consentiu

🔍 Checklist

  • Data deletion implementado? ❌
  • Data export implementado? ❌
  • PII detection? ❌
  • Retention policy? ⚠️
  • Consent management? ❌
