Jobs Page

Visualize, criar e gerenciar todos os jobs da fila. Acompanhe status, filtre por critérios eexecute ações em massa.

Layout da Página

A página de Jobs (/src/pages/Jobs.tsx) é composta por trs seções principais:

Header

Título "Jobs", botão "Create Job" e indicador de refresh quando há atualizações em andamento.

UI

Create Job Form

Formulário modal inline para criar novos jobs com todas as configurações disponíveis.

Feature

Filters

Filtros por status, fila e datas com debounce para evitar requests excessivos.

Logic

Jobs Table

Tabela com todos os jobs, badges de status coloridos e ações por linha.

Data

Header

O header da página contém o título e os controles principais. Quando há atualizações em background (refetch), um spinner giratório do Loader2 icon aparece ao lado do botão.

Create Job Form

O formulário de criação de job aparece quando o usuário clica em "Create Job". É um card com multipleis campos organizados em grid responsivo.

ℹ
Modal inline

Em vez de um modal separado, o formulário aparece como card inline dentro da página para melhor UX e contexto.

Campos do Formulário

1. Queue (Select)

Campo obrigatório que permite selecionar a fila onde o job será criado. Dados vindos do hook useQueues().

2. Webhook URL + Method

URL do webhook que receberá o POST/PUT/etc com o payload. Inclui um select lateral para o método HTTP:

  • GET - payload via query parameters
  • POST - payload no body JSON
  • PUT - payload no body JSON
  • PATCH - payload no body JSON
  • DELETE - payload via query parameters
