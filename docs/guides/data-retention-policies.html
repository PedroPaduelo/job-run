TabelaRetençãoJustificativa

QueueIndefinidaConfig

🗑️ Cleanup Automático

// Cron diário (00:00)
DELETE FROM job_logs WHERE created_at < NOW() - INTERVAL '90 days';
DELETE FROM audit_logs WHERE created_at < NOW() - INTERVAL '7 years';
VACUUM ANALYZE job_logs;

📦 Archival

Antes de deletar, exportar para S3/Google Cloud Storage em Parquet/CSV.

✅ Recomendações

  1. Cronjob diário — rodar cleanup às 00:00
  2. Partitioning por mês — facilita DROP partition
  3. Export antes deletar — manter archival por 10 anos
  4. pg_cron — agendar dentro do próprio PostgreSQL
