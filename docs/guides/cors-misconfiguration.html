CORS Misconfiguration

Configuração permissiva permite ataques cross-origin

🔓 O Problema

CORS config usa origem dinâmica:

const allowedOrigins = (process.env.ALLOWED_ORIGINS || 'http://localhost:5173,http://localhost:3000').split(',').map(o => o.trim());

await fastify.register(cors, {
  origin: (origin, callback) => {
    if (!origin) return callback(null, true);  // ← permite null origin (apps移动, curl)
    if (allowedOrigins.includes(origin) || process.env.NODE_ENV === 'development') {
      return callback(null, true);
    }
    callback(new Error('Not allowed by CORS'), false);
  },
  credentials: true,
});
🌐
Permissivo demais

if (!origin) return callback(null, true) permite requisições sem Origin (curl, mobile apps, server-to-server). Isso pode ser explorado.

💥 Riscos

  • CSRF via CORS — attacker faz fetch de API de site malicioso se credentials são enviados
  • Leak de dados — se ALLOWED_ORIGINS não está configurado, attacker pode ler responses
  • Wildcard + credentials — Access-Control-Allow-Origin: * com credentials: true é bloqueado pelo browser, mas se códigoallow all origin em dev, pode afetar production
  • Subdomain takeover — se allowedOrigins inclui subdomains curinga (*.exemplo.com), attacker pode registrar evil.exemplo.com

✅ Recomendações

  1. Não permitir null origin em produção: 
    origin: (origin, cb) => {
  if (!origin && process.env.NODE_ENV === 'production') {
    return cb(new Error('Origin required'), false);
  }
  // ... resto
}
  2. ValidarALLOWED_ORIGINS strictamente — sem wildcards, lista explícita HTTPS only
  3. Separar allowedOrigins por ambiente — produção vs staging
  4. Log rejected origins — para detectar scan/attack
  5. credentials: true apenas se necessário (API keys em headers, não cookies) — pode setar false se usar Bearer apenas
  6. HSTS — garantir HTTPS apenas

🔍 Checklist

  • ALLOWED_ORIGINS tem HTTPS? ⚠️
  • null origin bloqueado em prod? ❌
  • Wildcards usados? ❌
  • credentials: true necessário? ⚠️ (Bearer token não precisa de credentials)
  • Rejected origins são logados? ❌
