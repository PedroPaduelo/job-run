Dashboard (Página Principal)

Visão geral em tempo real do sistema de processamento de jobs

Visão Geral

A página Dashboard é a página inicial do frontend JobSync. Ela fornece uma visão completa do estado atual do sistema, incluindo estatísticas agregadas, timeline de atividade, jobs recentes e status das filas.

Componente

src/pages/Dashboard.tsx

2. Real-time Updates

O Dashboard utiliza 3 hooks de dados em paralelo, cada um com seu intervalo de refetch:

Intervalos de Refetch

Stats
REFETCH_INTERVALS.STATS = 30000
30 segundos
Timeline
REFETCH_INTERVALS.TIMELINE = 60000
60 segundos (1 min)
Jobs List
REFETCH_INTERVALS.JOBS_LIST = 30000
30 segundos

2.1 Hook useOverviewStats()

Busca estatísticas agregadas do sistema. Retorna objeto com:

const { data: stats, isLoading: statsLoading } = useOverviewStats();

// Estrutura de stats:
{
  totalJobs: number;
  todayJobs: number;
  processing: number;
  avgDuration: number;     // em ms
  failedToday: number;
  queues?: Array<{        // opcional, pode ser vazio
    id: string;
    name: string;
    waiting: number;
    active: number;
    failed: number;
  }>;
}

2.2 Hook useTimeline(24)

Busca dados de timeline para as últimas N horas (24 por padrão).

const { data: timeline } = useTimeline(24);

// Estrutura de timeline:
[
  {
    hour: string;         // timestamp ISO por hora
    total: number;
    completed: number;
    failed: number;
  },
  ...
]

2.3 Hook useJobs({ limit: 5 })

Busca lista paginada de jobs mais recentes.

const { data: recentJobs } = useJobs({ limit: 5 });

// Estrutura:
{
  jobs: Array<{
    id: string;
    queueName: string;
    status: JobStatus;
    createdAt: string;    // ISO timestamp
    // ... outros campos
  }>;
}

3. Gráfico Timeline (Recharts)

O gráfico usa ResponsiveContainer do Recharts para se adaptar ao tamanho do card.

3.1 Componente LineChart

<ResponsiveContainer width="100%" height="100%">
  <LineChart data={timeline}>
    <XAxis
      dataKey="hour"
      tickFormatter={(value) =>
        new Date(value).getHours().toString().padStart(2, "0")
      }
      fontSize={12}
    />
    <YAxis fontSize={12} />
    <Tooltip
      labelFormatter={(value) =>
        new Date(value).toLocaleString()
      }
    />

    {/* Série: Total */}
    <Line
      type="monotone"
      dataKey="total"
      stroke="#8884d8"
      strokeWidth={2}
      dot={false}
    />

    {/* Série: Completed */}
    <Line
      type="monotone"
      dataKey="completed"
      stroke="#82ca9d"
      strokeWidth={2}
      dot={false}
    />

    {/* Série: Failed */}
    <Line
      type="monotone"
      dataKey="failed"
      stroke="#ff6b6b"
      strokeWidth={2}
      dot={false}
    />
  </LineChart>
</ResponsiveContainer>

3.2 Cores das Séries

  • total: roxo - todos os jobs criados no período
  • completed: verde - jobs finalizados com sucesso
  • failed: vermelho - jobs que falharam

6. Estados de Carregamento (Loading)

Apenas o estado statsLoading é verificado para mostrar tela de carregamento inicial:

const { data: stats, isLoading: statsLoading } = useOverviewStats();
const { data: timeline } = useTimeline(24);
const { data: recentJobs } = useJobs({ limit: 5 });

if (statsLoading) {
  return <div className="flex items-center justify-center h-64">Loading...</div>;
}

Enquanto timeline ou recentJobs carregam, seus respectivos cards mostram fallbacks.
